AyurCare – Ayurvedic Hospital Management System

Overview

AyurCare is a console-based hospital management system made in C. It helps Ayurvedic hospitals and
Panchakarma centers manage their daily work. It stores data using file handling so the information
stays safe even after the program closes.
The system makes it easier to handle patients, doctors, therapists, medicines, therapy schedules, and
billing.

Features

• Easy menu-based interface
• Add, view, update, and delete patient records
• Manage doctor and therapist details
• Track medicine inventory
• Schedule Panchakarma therapies
• Create bills and view reports
• Store all data using C file handling

Main Menu Options

1. Patient Management
2. Doctor & Therapist Management
3. Medicine Inventory
4. Panchakarma Therapy Schedule
5. Billing
6. Reports
7. Exit
   
Program Flow

START → Show Menu → User Chooses Option → If option == 7 exit, else run module → Return to
Menu

Requirements

• A computer with a C compiler (GCC, MinGW, etc.)
• Any C IDE (CodeBlocks, Dev-C++, VS Code, etc.)

How to Run

1. Clone or download this project
2. Open the code in your IDE
3. Compile the .c file(s)
4. Run the executable
5. Use the menu to explore the system
